📖 Como funciona

💰 Para participar

Valor: R$ 30,00 por pessoa.
Encaminhe o comprovante de pagamento no grupo do Bolão para confirmar a participação.

🏆 Premiação

Valor total a definir de acordo com o número de participantes.
🥇 1º lugar: 40% · 🥈 2º lugar: 30% · 🥉 3º lugar: 20% · 4º lugar: 10%
🐢 Premiação especial para o último colocado (que tenha participado).

Para participantes

1. Abra este arquivo no navegador (funciona no celular).
2. Vá em Meus Palpites, digite seu nome e preencha os placares.
3. Clique em Copiar código e envie o código ao organizador (WhatsApp, e-mail...).
4. Quando o organizador reenviar o arquivo atualizado, abra para ver o Ranking e os jogos.

Para o organizador

1. Na aba Admin, cole os códigos recebidos e importe.
2. Lance os resultados conforme os jogos acontecem.
3. No mata-mata, defina os confrontos e reenvie o arquivo para nova rodada de palpites.
4. Use Gerar arquivo atualizado e mande no grupo — é só isso.

Pontuação

Fase de grupos: 🎯 acertou resultado e placar: 3 pts · ✅ acertou o resultado: 1 pt.
Mata-mata: 🎯 acertou resultado e placar: 5 pts · ✅ acertou o resultado: 2 pts. Vale o placar dos 90 min — empate é válido.
Desempate: quem tiver mais placares exatos. Palpites dos outros só ficam visíveis depois que o jogo começa.
